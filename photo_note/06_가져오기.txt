외부 이미지 가져오기
-> 포토샵 메뉴를 이용하거나
윈도우의 파일 탐색기(윈+e)를 통해 가져올 수 있다.

File 메뉴

open: 새 창으로 열기

place embedded(포함): 열린 창으로 가져오기
- 바운딩 박스(Bounding Box) -> 변형(Transform)
- Smart Object Layers(벡터)
- 메뉴를 사용하면 이미지를 하나만 가져올 수 있다.
단, 파일 탐색기를 이용하면 여러 개의 파일을 가져올 수 있다. 

파일 탐색기를 이용할 경우
- 창 안쪽으로 드래그하면 place
- 창 바깥쪽으로 드래그하면 open


* 이미지 캡처하기
윈도우 캡처 -> 윈+shift+s
알캡처

캡처한 결과는 클립 보드에 저장된다.
클립 보드란? 윈도우 임시 저장 공간
윈도우 캡처 시 1개의 이미지만 저장 가능
